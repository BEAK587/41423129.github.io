w3： 什麼是 git, GitHub, gist 與 cmsimde? 如何使用這些工具？為什麼要使用這些工具？

Git - 時光機 + 分工本

裝在你電腦裡的軟體，用來記錄檔案（尤其是程式碼）的「每一個修改歷史」。可以回到任何一個過去版本，也可以讓多人同時修改不同部分再合併。

GitHub - 程式碼的雲端硬碟 + 協作平台

一個網站，專門用來存放用 Git 管理的專案。讓你可以把本地專案備份到雲端，並方便地與他人協作、討論、分享。

Gist - GitHub 的便利貼

GitHub 提供的一個小功能，用來快速分享「一小段程式碼」或筆記。它背後也是一個迷你 Git 倉庫。

CMSimde - 簡單的網站製造機

一個非常輕量、操作簡單的內容管理系統，特別適合老師和學生用來快速建立課程網站、寫筆記。

如何使用這些工具？

步驟 1：安裝與設定 Git
步驟 2：Git + GitHub 基礎協作流程
步驟 3：使用 Gist 分享片段
步驟 4：使用 CMSimde 建網站
為什麼要使用這些工具？

用 Git 在本地管理程式碼版本，然後推送到 GitHub 進行備份和分享。如果想只分享一段代碼，就用 Gist。如果你想快速做一個教學網站來放這些內容，就可以用 CMSimde。

w7：Brython 是什麼？

Brython（Browser Python）

是在瀏覽器中執行的 Python 3 實現。用 Python 替代 JavaScript 進行前端開發。

如何在瀏覽器配置 Brython 環境？

第 1 步：建立 HTML 檔案
第 2 步：加入 Brython CDN
第 3 步：啟動 Brython
第 4 步：編寫 Python 程式

Brython 能做什麼？

1. DOM 操作（取代 jQuery）
2. AJAX 請求（取代 fetch/axios）
3. 繪圖與動畫
4. 與 JavaScript 互操作
5. Python 標準庫支援
6. 建立 GUI 應用

w10：如何在瀏覽器中模擬機器人作動？

機器人控制程式概念

機器人控制程式碼範例
# 記錄機器人當前狀態
position = (0, 0)
direction = "right"
carrots_collected = 0

def 移動():
    # 根據方向更新位置
    global position
    x, y = position
    
    if direction == "right":
        x += 1
    elif direction == "left":
        x -= 1
    elif direction == "up":
        y += 1
    elif direction == "down":
        y -= 1
    
    position = (x, y)
    print(f"移動到位置: {position}")

def 轉彎():
    # 改變方向
    global direction
    
    if direction == "right":
        direction = "down"
    elif direction == "down":
        direction = "left"
    elif direction == "left":
        direction = "up"
    elif direction == "up":
        direction = "right"
    
    print(f"轉向: {direction}")

def 採收():
    # 增加胡蘿蔔數量
    global carrots_collected
    carrots_collected += 1
    print(f"採收到胡蘿蔔! 總計: {carrots_collected}")

def 前方有胡蘿蔔():
    # 檢查前方是否有胡蘿蔔（模擬函數）
    return True  # 實際應用中這裡會有判斷邏輯

# 重複執行直到完成
還有胡蘿蔔 = True
步數限制 = 20
當前步數 = 0

while 還有胡蘿蔔 and 當前步數 < 步數限制:
    
    # 檢查前方是否有胡蘿蔔
    if 前方有胡蘿蔔():
        採收()
    
    # 移動到下一個位置
    移動()
    
    # 每4步轉彎一次（模擬巡邏模式）
    if 當前步數 % 4 == 0:
        轉彎()
    
    當前步數 += 1
    
    # 更新狀態（模擬條件）
    if carrots_collected >= 5:
        還有胡蘿蔔 = False
        print("任務完成! 已收集足夠的胡蘿蔔。")
    
    if 當前步數 >= 步數限制:
        print("達到步數限制，停止執行。")
        break

print(f"最終結果 - 位置: {position}, 方向: {direction}, 胡蘿蔔: {carrots_collected}")

程式說明

狀態變數： position（位置）、direction（方向）、carrots_collected（收集的胡蘿蔔數量）
移動函數： 根據當前方向更新機器人位置
轉彎函數： 改變機器人方向（右→下→左→上→右）
採收函數： 增加收集的胡蘿蔔數量
主迴圈： 重複執行直到完成任務或達到步數限制
學習重點：

這個程式展示了如何用 Python 控制機器人的基本行為。重點在於理解狀態管理、條件判斷和迴圈控制。實際應用時，前方的胡蘿蔔檢查函數會根據感測器數據回傳真實的判斷結果。
